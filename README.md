# 体彩福彩记账工具

#### 介绍
辅助Lott店老板记录信息的一整套工具。具体功能如下：
1、目前支持的彩种包括：竞彩足球，竞彩篮球，北京单场，排列3，排列5，任选九，足球14场，七星彩，大乐透，快乐8，福彩3D，七乐彩，双色球。地方性在开发中；
2、同步彩种相关信息，供用户下单；
3、记录用户下单数据、获取对应结果数据、判断是否命中及预计奖励数额；
4、用户可分享自己研究的方案，供其他用户参考。也可选择一起购买，共担风险共享收益；
5、店主综合管理所有订单，查看内容并操作；
6、分类统计，供店主对账；
